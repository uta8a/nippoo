# timezLog

日報 generated by Toggl

## 構成

```sh
deno task run
```

で、以下のようにディレクトリを作っておくとデータを配置する。

```text
data/
  json/
    2022-07-24.json
  md/
    2022-07-24.md
  pub_html/
    index.html # list
    2022-07-24.html
```
